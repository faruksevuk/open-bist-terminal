# Ekran görüntüleri

Ana README bu klasördeki üç görseli referanslıyor. Uygulama çalışırken (localhost:4000)
şunları çek ve buraya bu adlarla kaydet:

| Dosya | Sayfa | Ne göstermeli |
|-------|-------|---------------|
| `dashboard.png` | `http://localhost:4000` | Ana panel — "Bugün Ne Yapmalı" + setup kartları + fırsatlar (veri dolu haliyle) |
| `settings.png`  | `http://localhost:4000/settings` | AI sağlayıcı seçici + risk profili + faktör ağırlıkları + alttaki kaydet-barı |
| `ticker.png`    | `http://localhost:4000/ticker/THYAO` | Hisse detayı — mum grafiği + indikatörler + fundamentaller |

İpucu: koyu tema varsayılan; net görünmesi için pencereyi ~1280px genişlikte tut.
Windows'ta `Win+Shift+S`, macOS'ta `Cmd+Shift+4` ile bölge seçip kaydedebilirsin.
